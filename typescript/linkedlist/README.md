# Doubly Linked List (TypeScript)

Doubly linked list implementation written in TypeScript as part of the
DSA Multilang project.

Includes a small test driver used to validate core list operations.

---

## Files

- `linkedlist.ts` – Doubly linked list implementation
- `test.ts` – Simple driver to exercise the list

---

## How to Run

From the linked list directory:

```bash
cd DSA-MULTILANG/typescript/linkedlist
npx tsc linkedlist.ts test.ts
node test.js
