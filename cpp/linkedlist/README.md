# Linked List (C++)

Singly linked list implementation written in modern C++.

## Features

- Append / prepend
- Insert and delete at index
- Delete head / tail
- Search
- Copy constructor and assignment operator
- O(1) size tracking

## Files

- `linkedlist.h` – Class interface and documentation
- `linkedlist.cpp` – Implementation
- `test.cpp` – Interactive driver for testing list operations
- `Makefile` – Build instructions

## Build & Run

```bash
make
./test
