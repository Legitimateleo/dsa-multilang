# Linked List (Python)

Singly linked list implementation written in Python as part of the **DSA Multilang** project.

This implementation mirrors the behavior and API of the C++ version to reinforce understanding of data structures across languages.

---

## Features

- Append and prepend elements
- Delete head and tail nodes
- Insert and delete at a given index
- Search for a value
- O(1) size tracking using an internal counter
- Clean string representation via `__str__`

---

## Files

- `linkedlist1.py` – Linked list and node implementation
- `test.py` – Simple driver script to verify behavior
